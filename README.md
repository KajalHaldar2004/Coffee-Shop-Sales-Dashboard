# Coffee-Shop-Sales-Dashboard
This project is an Excel-based dashboard created using a coffee shop sales dataset from Maven Analytics (Jan–Jun 2023). The purpose of the project is to analyze sales performance and identify useful insights that can help in business decision-making.

Work Done:
I started with data cleaning and transformation using Power Query. This included removing inconsistencies, trimming text columns, and creating additional columns where required. I created a conditional column to categorize product sizes and a custom column to calculate total revenue using unit price and quantity, as it was not present in the dataset. I also extracted time-related fields such as hour, day, and month to analyze sales trends and peak hours.

For analysis, I used Power Pivot to create measures like total sales, total quantity, and average order value. This helped in summarizing the data efficiently.

Dashboard:
The dashboard is built using pivot tables, charts, and slicers. It shows sales trends over time, peak hours, monthly performance, and best-selling products. The slicers allow filtering the data for better analysis.

Key Insights:
1. Sales are highest during morning hours, indicating strong breakfast-time demand.

2. Coffee products contribute the most to revenue, making them the core category.

3. Items like Barista Espresso and Brewed Coffee are top-selling products.

4. Large and Regular sizes are more popular than Small.

5. Sales remain fairly consistent across weekdays, showing stable demand.

6. All store locations are performing well, with only slight variations.”

Conclusion:
This project helped me understand how to clean data, create meaningful metrics, and build a simple interactive dashboard in Excel. It also shows how raw data can be converted into insights that support better business decisions.
